ASCII Chess

Made with: Python
Source: GitHub

ASCII Chess is a project I worked on after taking my first CS class at Stanford: CS106A Programming Methodology. I wanted to step away from adding external libraries and focused on implementing all the logic and graphical display to just base Python. This game allows 2 players to play chess and performs its own checks throughout the game for board states. When a check or checkmate occurs, The game is aware of it and acts accordingly (preventing illegal moves, declaring game end, etc.)

This project acted as a great base to understanding how to implement game logic, piece movement, and reading board states. I hope to create a GUI for this game and integrating it with the web
