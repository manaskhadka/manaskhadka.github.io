Discord Bot

Made with: JavaScript, Python, discord.py, discord.js, BeautifulSoup, heroku
Source: GitHub

Making a Discord Bot started off as a small project but quickly grew in scale. It was a summer project I made as something that would be fun for me and my friends to use on our Discord server - it was silly and truly a passion project. At first, it was only a chat bot that would return text based on certain commands, but I kept adding features over time.

These features include:

A Magical 8 Ball Game

Input a response and you get a random answer out

Weather Report

fetches weather and time data for an inputted city

Urban Dictionary Webscraper

displays the top response for an inputted search phrase from urbandictionary.com

Wikipedia Webscraper

displays the article for an inputted search phrase from wikipedia.org
